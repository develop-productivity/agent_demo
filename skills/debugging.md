# Debugging

## Principle

Never guess.

Gather evidence first.

## Workflow

### Reproduce

Confirm issue exists.

### Observe

Collect:

- logs
- stack traces
- error messages

### Localize

Find:

- component
- module
- function

### Hypothesize

Generate possible causes.

### Verify

Test one hypothesis at a time.

### Fix

Apply minimal change.

### Validate

Confirm:

- original issue fixed
- no regression introduced

## Rules

Never apply multiple unrelated fixes simultaneously.

Always explain root cause.
