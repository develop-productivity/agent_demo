# Test Runner

## Before Change

Identify:

- existing tests
- test framework

Examples:

- jest
- vitest
- mocha
- pytest

## After Change

Run:

- affected tests
- integration tests if needed

## Validation

Check:

- compilation
- lint
- unit tests

## Output

Report:

- tests executed
- passed
- failed
- skipped
